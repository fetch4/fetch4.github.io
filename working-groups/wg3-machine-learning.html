Working Group 3: Machine Learning

WG3 is building the computational infrastructure that will make FETCH4 data scalable, accessible, and operational. This group develops machine learning emulators, satellite proxies, and data assimilation tools to support real-time methane monitoring and historical analysis.

Full chemistry-climate models are comprehensive descriptions of the atmosphere, but computationally expensive. This makes them difficult to use for real-time decision-making or iterative data analysis. WG3 is addressing this bottleneck by creating machine learning surrogates that retain physical interpretability while allowing for rapid simulation and assimilation of new data.

WG3 has developed the first linear inverse model (LIM) of a coupled chemistry-climate model, using training data from the GFDL-CM3. This emulator accurately reproduces spatio-temporal variations in OH and can be used for efficient chemical data assimilation.

In parallel, WG3 is developing deep learning emulators of chemical mechanisms. Graduate students at the University of Washington are training these models using outputs from the GISS and CESM models. WG3 is also building a new box modeling framework to evaluate the effects of halogen chemistry on methane isotopes.

Satellite data efforts are advancing rapidly. This group is using multispectral imagery from Landsat and Sentinel satellites to construct a long-term record of methane emissions from oil and gas infrastructure. A new U-Net-based technique for image inpainting is being used to recover missing Landsat 7 data from the early 2000s. This allows for analysis of methane trends during a critical period of post-Soviet industrial activity.
